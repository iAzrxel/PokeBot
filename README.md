# PokéBot

Strategic Pokémon Showdown bot built with Python featuring battle evaluation, scouting, opponent memory and adaptive decision-making.

---

## Features

* Move evaluation system
* Switch evaluation logic
* Opponent memory
* Basic scouting mechanics
* Adaptive battle decisions
* Force switch handling
* Strategic score-based AI
* Modular architecture
* Built for Pokémon Showdown

---

## Tech Stack

* Python
* Pokémon Showdown API / environment
* Object-oriented architecture
* Score-based battle evaluation system

---

## Installation

```bash
git clone <repository-url>
cd <repository-name>
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

### Windows

```bash
venv\Scripts\activate
```

### Linux / MacOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running

```bash
python src/main.py
```
